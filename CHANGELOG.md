# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]
- No unreleased changes so far

## [0.4.2] - 2026-01-23
### Fixed
- Improved reliability on recording trips and charging sessions

## [0.4.1] - 2026-01-20
### Fixed
- Fixes a problem where trips were recorded without a start location when the vehicle was parked before more than 5 minutes

## [0.4] - 2026-01-19
### Fixed
- Fixes a problem where not multiple vehicles could be recorde to the database

## [0.3] - 2026-01-11
### Changed
- Improved robustness
- Prepared infrastructure to store data in correct unit based on locale

## [0.2] - 2026-01-08
### Added
- Recording of battery temperatures
- Recording of refueling of combustion vehicles

### Fixed
- Fixes bug where charging session start level was overwritten
- Fixes bug where charging session start date was overwritten

## [0.1] - 2026-01-04
Initial release, let's go and give this to the public to try out...
